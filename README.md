# BenchPilot

Agent-first embedded device lifecycle toolkit.

## Status

Version 0.0.0 is the project initialization release. There is currently no
usable API or CLI.

BenchPilot is intended to grow toward:

- embedded device discovery
- build and flash lifecycle
- serial log collection
- multi-device orchestration
- agent-friendly structured results

Repository: https://github.com/RavelloH/benchpilot
